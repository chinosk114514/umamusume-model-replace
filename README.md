# umamusume-model-replace
- replace umamusume models



# 使用方法

- 安装`Python3.8及以上版本`
- 安装依赖: `pip install -r requirements.txt`

```
UnityPy~=1.8.15
```

- 运行`main.py`

```shell
python main.py
```

- 根据提示操作即可。
