# umamusume-model-replace
- replace umamusume models



# 使用方法

- 安装`Python3.8及以上版本`
- 安装依赖: `pip install -r requirements.txt`

```
UnityPy~=1.8.15
```

- 运行`main.py`

```shell
python main.py
```

- 根据提示操作即可。

```
[11] 更换头部模型\n
[12] 更换身体模型
[13] 更换尾巴模型(不建议)
[14] 更换头部与身体模型
[15] 更换抽卡开门人物	
[21] 修改角色身体贴图
[22] 修改角色头部贴图
[23] 修改支援卡贴图
[24] 修改支援卡缩略图贴图
[25] 批量修改支援卡
[3]  更换技能动画
[4]  更换G1胜利动作(实验性)
[51] Live服装解锁
[52] 清除Live所有模糊效果
[9]  复原所有修改
[91] 清理 backup,edited 文件夹
[0] 退出
```
